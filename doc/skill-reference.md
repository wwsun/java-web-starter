# Skill Reference

## 典型工作流串联

- 启动：/session-start → /understand → /scaffold
- 需求：/office-hours → /plan-ceo-review → /create-todos
- 设计：/autoplan（三视角一键联合评审）
- 编码循环：/implement → /test → /review → /checkpoint
- 前端：/design-shotgun → /frontend-design → /browse → /qa
- 上线：/score-app → /ship → /document-release
